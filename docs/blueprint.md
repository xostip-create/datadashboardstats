# **App Name**: BarBook

## Core Features:

- Sales Logging: Record each sale with item, quantity, and price.
- Opening Stock Input: Record the starting quantity of each item at the beginning of the day.
- Closing Stock Input: Record the remaining quantity of each item at the end of the day.
- Sales Summary: View a summary of total sales for the day, broken down by item.

## Style Guidelines:

- Primary color: Deep, rich burgundy (#800020) to evoke a sense of warmth and sophistication.
- Background color: Soft, desaturated beige (#F5F5DC) to provide a neutral backdrop.
- Accent color: Muted gold (#D4A27A) for highlights and important actions, creating a luxurious feel.
- Font: 'Alegreya', a humanist serif that gives an elegant, intellectual, contemporary feel; suitable for both headlines and body text.
- Use simple, line-based icons to represent different items and actions. Consistent in style to ensure clarity.
- A clear and well-organized layout with intuitive navigation to make data entry and review easy.
- Subtle transitions and animations to provide feedback on user actions, creating a polished user experience.
